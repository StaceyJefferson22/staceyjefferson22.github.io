CSCE 242: Web Application

Stacey Allen Jefferson Jr.

CSCE 242 is a web development course instructed by Portia Plante. It involves students mastering HTML and CSS to create the front-end of a website and later JavaScript to give it functionality. The class composes of assignments, in-class assignments, a midterm project, and a final project. Face-to-face meetings occur Tuesdays and Thursdayat 11:40-12:55pm.

CSCE 242 is very similiar to CSCE 190. They are both web development courses that utilize HTML and CSS. However, 242 is a three-hour long course whereas 190 is a one-hour course. Also, 242 includes JavaScript, which is not utilize in 190.
